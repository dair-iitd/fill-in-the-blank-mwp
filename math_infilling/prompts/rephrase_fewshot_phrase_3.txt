Q: There are 15 trees in the grove. _____. After they are done, how many trees would be there?
A: 21
Guess: x trees were added.
Rephrased: There are 15 trees in the grove. x trees were added. After they are done, there would be 21 trees. Find the value of x.
Answer: There are 15 trees originally, Then there were 21 trees after some more were planted. So there must have been x = 21 - 15 = 6 trees. The answer is 6.

A: 5
Guess: x more cars arrive

Answer: There are originally 3 cars. x more cars arrive. 3 + x = 5, so x = 5 - 3 = 2. The answer is 2.

Q: Leah had 32 chocolates and _____. If they ate 35, how many pieces do they have left in total?
A: 39
Guess: her sister had x 
Rephrased: Leah had 32 chocolates and her sister had x. If they ate 35, they had 39 pieces left in total. Find the value of x.
Answer: Originally, Leah had 32 chocolates. Her sister had x. So in total they had 32 + x. After eating 35, they had 32 + x - 35 = x - 3. Since they had 39 chocolates left, x - 3 = 39, so x = 39 + 3 = 42. The answer is 42.

Q: There were nine computers in the server room. _____, from monday to thursday. How many computers are now in the server room?
A: 29
Guess: x computers were added
Rephrased: There were nine computers in the server room. x computers were added, from monday to thursday. There are 29 computers now in the server room. Find the value of x.
Answer: There were originally 9 computers. For each of 4 days, x more computers were added. Since the total number of computers is 29, 9 + 4 * x = 29. So 4 * x = 29 - 9 = 20, and x = 20 / 4 = 5. The answer is 5.
